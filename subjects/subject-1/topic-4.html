Computer Hardware
  • refers to the physical equipment used for the input, processing, output and storage activities of a computer system

    • Classes of Computers

    supercomputers
    indicates the fastest computers available at a given time
    used by large organizations to execute demanding tasks for large data

    mainframe
    specialize in transaction processing and business applications
    examples are payroll programs, Amazon, and student grade calculation

    desktop pcs
    a standard tool for business and the home
    generally includes a CPU, monitor, keyboard, storage

    laptops/notebooks
    small, easily transportable, lightweight microcomputers
    provide access to processing power and data outside an office environment

    tablets
    a complete computer contained entirely in a flat touch screen that users operate via a stylus, digital pen or fingertip instead of a keyboard or mouse

    thin client systems
    a desktop system without the full functionality of a PC are less complex, easier and less expensive to operate and support than PCs

    Hardware Components

    the CPU
    The central processing unit
    performs the computation or ‘number crunching’ inside computers
    is a microprocessor of millions of transistors embedded in a circuit

    control unit
    - decodes program instructions and controls the data flow between the ALU, registers, caches, storage, output devices

    arithmetic-logic unit (ALU)
    - performs the mathematic calculations and makes logical comparisons

    registers
    - are high-speed storage areas that store very small amounts of data for short periods

    Primary Storage
    also known as main memory
    the four main types of primary storage:

    Register
    - have the least capacity. stores extremely limited data

    Cache Memory
    - have the least capacity. stores extremely limited data

    Random Access Memory (RAM)
    - holds small amounts of data for processing
    - is volatile — content is lost if the current is lost / turned off
    - when a software program is opened, it is brought from secondary storage into RAM. as the program is used, the program’s data are sent into the registers, then to the CPU

    Read-Only Memory (ROM)
    - nonvolatile chip holding critical instructions that can only be read by the computer and cannot be changed by the user (booting, etc.)

    Secondary Storage
    nonvolatile storage designed to store very large amounts of data for extended periods
    takes more time to retrieve data from it than from RAM

    magnetic tape
    - the cheapest type medium that can handle large amounts data but is the slowest at data retrieval

    magnetic disks (hard drives)
    - the most commonly used storage devices because of low cost, high speed, and large capacity

    solid state drives (SSDs)
    - use less power, are silent and faster than hard drives, but cost significantly more

    flash memory devices
    - or memory cards. use less power, smaller, more durable than hard drives but store less data (e.g. USB)

    read-only memory (ROM)
    - nonvolatile chip holding critical instructions that can only be read by the computer and cannot be changed by the user (booting, etc.)

    Input Devices
    allow people and other technologies to enter data into a computer The two main types of input devices:

    human data-entry devices
    - require human effort to input data
    - e.g. keyboard, mouse, pointing stick, trackball, joystick, touchscreen, stylus and voice recognition

    source-data automation devices
    - input data with minimal human intervention; speeds up data collection and reduces errors
    - e.g. barcode scan, sensors, magnetic strip readers, OCR

    Output Devices
    the output generated by a computer can be transmitted to the user via several output devices and media
    types of output devices:

    monitors
    e.g. cathode ray tubes, LCD, flexible displays

    printer
    e.g. laser, inkjet, thermal

    plotters
    - computer-directed pens for creating high-quality images

    e-book readers
    - wireless, portable reading device with access to books

    Computer Software
  • refers to the programs and applications that give instructions to the hardware components for specific tasks
  • enables the user to instruct a computer system to perform specific functions

    • Types of Software

    Systems software
    - an intermediary between computer hardware and application programs
    - provides commonly used sets of instructions for all applications
    - directs the basic functions of the computer (e.g. loading when turned on)
    - controls and supports information-processing activities
    - facilitates the programming, testing and debugging of programs

    Operating system (OS)
    is the director of a computer’s operations
    provides an interface between the user and the hardware

    Application software
    - consists instructions that direct a computer system to perform specific information-processing activities which provide functionality for users
    - applications that increase productivity, aka personal application software:

    spreadsheets
    is the director of a computer’s operations
    provides an interface between the user and the hardware

    wordprocessing
    is the director of a computer’s operations
    provides an interface between the user and the hardware
